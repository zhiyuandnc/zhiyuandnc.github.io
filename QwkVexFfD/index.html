Java中所有TimeZoneID

· 2021-01-10 · # timezone # 时区 # Java

一、使用代码输出Java中所有TimeZoneID

public static void main(String[] args) {
    Set<String> zoneIds = ZoneId.getAvailableZoneIds();
    for (String zoneId : zoneIds) {
        System.out.println(zoneId);
    }
}
